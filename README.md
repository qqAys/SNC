# SNC

## 简介
SNC是一款修改MAC地址和主板UUID的软件。

所谓机器码是多种机器识别码的统称，包含：网卡物理地址(MAC)、主板序列号(Motherboard SYSTEM UUID/SN)等。

本软件使用Python编写，能够生成随机的、合规的物理地址(MAC)和主板UUID进行系统级修改，加入了免重启机制，兼容网吧、电竞酒店等无盘系统。
